O que é?

Middleware

Ver Fonte
Intermediário

Middleware é um software que atua entre aplicações, sistemas ou serviços, facilitando a comunicação entre eles.

Funcionalidades

Pode gerenciar autenticação, logs, compressão de dados, cache, segurança e muito mais.

Exemplo

Em um site, um middleware pode verificar se um usuário está autenticado antes de permitir o acesso a uma página.

Multer

Ver Fonte
Biblioteca para Uploads

Multer é um middleware do Node.js usado para processar uploads de arquivos em aplicações Express.

Trabalha com Formulários

Permite lidar com multipart/form-data, que é o formato usado para envio de arquivos via formulários HTML.

Armazenamento Flexível

Suporta armazenamento de arquivos no servidor (diskStorage) ou em serviços na nuvem, como Amazon S3.
